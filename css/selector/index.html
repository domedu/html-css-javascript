Attribute selector

  1. [data-value] { } Attribute exists
  2. [data-value="foo"] { } Attribute has this exact value
  3. [data-value*="foo"] { } Attribute value contains this value somewhere in it
  4. [data-value~="foo"] { } Attribute has this value in a space-separated list somewhere
  5. [data-value^="foo"] { } Attribute value starts with this
  6. [data-value|="foo"] { } Attribute value starts with this in a dash-separated list, eg: [data-years|="1990"] select the data-years="1990-2000" but not the data-years="1980-1990"
  7. [data-value$="foo"] { } Attribute value ends with this

hierarchy selector

  1. sibling selector: p + p {} select the directly after p element
  2. sbling selector: img ~ p {} select all the p after img element
  3. child selector: div > span {} select span after div
  4. descendant selector: div span {} select span in div
