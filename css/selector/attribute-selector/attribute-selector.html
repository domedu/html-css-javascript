7种属性选择器

    
    [data-value] {
    /* Attribute exists */
    }

    [data-value="foo"] {
    /* Attribute has this exact value */
    }

    [data-value*="foo"] {
    /* Attribute value contains this value somewhere in it */
    }

    [data-value~="foo"] {
    /* Attribute has this value in a space-separated list somewhere */
    }

    [data-value^="foo"] {
    /* Attribute value starts with this */
    }

    [data-value|="foo"] {
    /* Attribute value starts with this in a dash-separated list */
    }

    [data-value$="foo"] {
    /* Attribute value ends with this */
    }
[action]:有action属性
[action="refresh"]:action属性的值为refresh
[action="fa bar"]:属性值是以空格分开的多个值，bar是其中一个值
[action="fashion"]:属性值是以fa开始
[action="foo-icon"]:属性值是以icon结尾
[action="food"]属性值中包含oo
[action="en-fa-so"]属性值是以连字符连接,从左向右，以en开始
